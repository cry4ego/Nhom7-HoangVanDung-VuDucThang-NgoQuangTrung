# Luồng nghiệp vụ End-to-End

File: `Nhom07_BusinessFlow_QuanLyKhachHang_QuanLyCongViec.pdf`

Sơ đồ mô tả luồng xử lý một đơn hàng từ lúc khách hàng liên hệ đến khi được chăm sóc sau bán hàng, đi qua 4 module: `nhan_su` (HRM - dữ liệu gốc), `customer_management`, `task_management` và `chatbot_support`. Đánh dấu rõ điểm tích hợp HRM, các trigger tự động hóa (Mức 2) và điểm tích hợp AI/LLM + External API Telegram (Mức 3).
